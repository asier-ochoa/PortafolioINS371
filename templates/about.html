About me and this webpage

El sitio web fué escrito con html a mano. Le sugiero que haga Ctrl+Shift+C y mire el html para que vea la poca experiencia que tengo. El backend es un simple programa de python usando Flask y jinja2 para el preporcesamiento del html. En total el codigo python hace solo como 20 lineas. Si usted quiere probarlo en su maquina, yo uso Pipenv para manejar dependecias.

Al clonar el repo, necesitaria hacer "pipenv install flask" y luego "pipenv run python main.py". Obviamente necesita instalar python3, pip y pipenv previamente.

Como el sitio web se ve en mi buscador (Ungoogled Chromium 99)
La fuente usada para el texto es Inconsolara-dz, si no lo tiene instalado saldrá uno por defecto.
